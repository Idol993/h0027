## 1. 产品概述

宠物寄养预约与看护日记系统，为宠物主人提供便捷的寄养预约服务，为养护员提供专业的入住登记和日常看护记录工具，为管理员提供全面的运营数据管理能力。解决传统宠物寄养中信息不透明、沟通不及时、账目不清晰的痛点。

- 目标用户：需要短期寄养宠物的宠物主人、宠物寄养机构养护员、机构管理员
- 核心价值：预约透明化、看护可视化、异常即时化、账单自动化

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 宠物主人 | 手机号注册 | 提交预约、查看预约状态、查看看护日记、提交评价 |
| 养护员 | 管理员分配账号 | 扫码核销、入住登记、发布看护日记、记录异常事件、离店结算 |
| 管理员 | 系统预设账号 | 查看运营数据、导出报表、查询历史记录、管理用户和笼位 |

### 2.2 功能模块

1. **登录注册页**：角色切换、手机号登录、管理员账号登录
2. **机构首页**：机构信息展示、笼位概览、评价展示、快速预约入口
3. **预约寄养页**：日期选择、宠物信息填写、实时笼位查询、替代日期推荐
4. **我的预约页**：预约列表、状态筛选、预约详情入口
5. **预约详情页**：预约信息、看护日记时间线、异常事件记录、评价入口
6. **养护工作台**：今日待核销、在住列表、快捷日记入口
7. **入住登记页**：扫码核销、宠物状态登记、照片上传
8. **看护日记页**：日记发布（文字+图片）、历史日记查看
9. **异常记录页**：异常事件登记、推送状态跟踪
10. **离店结算页**：自动账单生成、确认接回
11. **管理面板页**：今日在住数、预约趋势、笼位使用率、平均评分
12. **报表查询页**：月度报表导出、历史记录多条件查询

## 3. 核心流程

### 3.1 预约流程

宠物主人选择寄养日期 → 系统查询笼位余量 → 有空位则锁定确认/无空位推荐替代日期 → 预约成功短信通知 → 主人随时查看预约状态

```mermaid
flowchart TD
    "选择寄养日期" --> "查询笼位余量"
    "查询笼位余量" --> "有空位?"
    "有空位?" -->|"是"| "锁定笼位确认预约"
    "有空位?" -->|"否"| "推荐3个替代日期"
    "推荐3个替代日期" --> "选择替代日期"
    "选择替代日期" --> "查询笼位余量"
    "锁定笼位确认预约" --> "短信通知主人"
    "短信通知主人" --> "预约完成"
```

### 3.2 入住与看护流程

入住当天养护员扫码 → 核销预约 → 登记宠物状态和照片 → 每日发布看护日记 → 异常时记录并推送 → 24h未查看则催促 → 连续未响应通知管理员

```mermaid
flowchart TD
    "入住当天扫码核销" --> "登记宠物状态"
    "登记宠物状态" --> "上传入住照片"
    "上传入住照片" --> "入住完成"
    "入住完成" --> "每日发布看护日记"
    "每日发布看护日记" --> "是否异常?"
    "是否异常?" -->|"是"| "记录异常事件"
    "记录异常事件" --> "立即推送主人"
    "立即推送主人" --> "24h内是否查看?"
    "24h内是否查看?" -->|"否"| "再次催促"
    "24h内是否查看?" -->|"是"| "结束"
    "再次催促" --> "是否响应?"
    "是否响应?" -->|"否"| "通知管理员"
    "是否响应?" -->|"是"| "结束"
    "是否异常?" -->|"否"| "继续看护"
```

### 3.3 离店与评价流程

宠物离店 → 系统按天数×日费率生成账单 → 养护员标记已接回 → 订单关闭 → 72h内主人可评价 → 评价展示在首页并参与星级计算

```mermaid
flowchart TD
    "宠物离店" --> "自动生成账单"
    "自动生成账单" --> "养护员确认接回"
    "养护员确认接回" --> "订单关闭"
    "订单关闭" --> "72h内可评价?"
    "72h内可评价?" -->|"是"| "主人提交评价"
    "72h内可评价?" -->|"否"| "超时不可评价"
    "主人提交评价" --> "评价展示在首页"
    "评价展示在首页" --> "参与星级计算"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暖珊瑚色 (#FF6B6B) 搭配奶油白 (#FFF8F0)，传递温暖关怀感
- **辅助色**：薄荷绿 (#4ECDC4) 用于成功/确认状态，柔和紫 (#A78BFA) 用于信息提示
- **警告色**：琥珀橙 (#F59E0B) 用于异常事件，红色 (#EF4444) 用于紧急通知
- **按钮风格**：圆角胶囊形按钮，主按钮带柔和阴影，hover时微微上浮
- **字体**：标题使用 Nunito（圆润友好），正文使用 Noto Sans SC（清晰中文）
- **布局风格**：卡片式布局，圆角大卡片，柔和阴影，留白充足
- **图标风格**：线性图标配合宠物主题插画元素
- **整体氛围**：温暖、亲切、专业，像宠物咖啡馆一样的舒适感

### 4.3 响应式设计

- 桌面端优先设计，最小宽度1280px
- 平板端（768px-1024px）：侧边栏收起为汉堡菜单，卡片改为单列
- 移动端（<768px）：底部Tab导航，卡片全宽，表格改为卡片列表

### 4.4 动效设计

- 页面切换：淡入+微上浮，时长300ms
- 卡片hover：微微上浮+阴影加深
- 笼位状态变化：脉冲动画
- 异常推送：红色呼吸灯效果
- 数据看板：数字滚动计数动画
- 评价星星：点击时放大弹跳
